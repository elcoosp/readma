# Tutorial Intro

Let's discover **Readma in less than 5 minutes**.

## Getting Started

Create a `readma.config.ts` file at the root of your project workspace.

Export default a config object like the
[following](https://github.com/elcoosp/readma/blob/main/readma.config.ts)

<CodeBlock language="ts" title="readma.config.ts">{readmaConfig}</CodeBlock>

:::warning
This file will be interpreted by a `deno` runtime.
:::

### What you'll need

- [Deno](https://deno.com/) version 2 or above

## Generate readme(s) of your project workspace

Generate a readme per project workspace member and/or a main one by running the
following command

```bash
deno run -A jsr:@readma/cli gen
```
